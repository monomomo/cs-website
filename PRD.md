# 路觅教育 Lumist - 计算机组师资展示网站 PRD

## 1. 产品概述

### 1.1 产品背景
路觅教育（Lumist）是一个线上计算机教育教培机构。随着业务发展，需要一个专业的网站展示计算机组的师资力量，让潜在学员和家长能够直观地了解教师的背景、经验和教学特色，从而增强对机构专业性的信任。

### 1.2 目标用户
- **主要用户**：正在选择计算机培训课程的潜在学员及其家长
- **用户诉求**：希望了解教师的专业背景、教学经验和真实评价，以做出选择决策

### 1.3 产品目标
- 提升机构专业形象，展示强大的师资力量
- 帮助用户快速了解每位教师的特点和擅长领域
- 通过真实学员评价增强可信度
- 作为招生的重要转化入口

### 1.4 成功指标
- 用户在页面平均停留时间 > 30秒
- 教师卡片点击率 > 50%
- 详情弹窗打开后完整阅读率 > 30%

## 2. 设计规范

### 2.1 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| Primary | `#FFC107` | 明黄色，品牌标识色 |
| Primary Light | `#FFD54F` | 浅黄色 |
| Primary Dark | `#FFA000` | 深黄色 |
| Background | `#0D0D0D` | 近黑色主背景 |
| Surface | `#1A1A1A` | 卡片背景 |
| Surface Light | `#252525` | 悬浮层背景 |
| Accent | `#00D4FF` | 科技蓝点缀 |
| Text Secondary | `#A0A0A0` | 次要文字 |

### 2.2 科技感设计元素

- **代码元素**：Hero背景代码雨效果
- **AI元素**：神经网络节点连线动画
- **几何装饰**：科技蓝渐变线条、六边形网格
- **动效**：装饰元素缓慢漂移/脉冲

### 2.3 专业度展示

- **数据量化**：累计学员、就业率、评分、竞赛获奖
- **公司背景**：前阿里巴巴/京东/字节跳动等
- **项目经验**：具体项目名称、角色、规模
- **技术链接**：GitHub / 博客 / 掘金
- **学术成就**：顶会论文、专利、竞赛金牌
